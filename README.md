# SuperMiroGame - Fiesta CIDE

Juego de plataformas 2D en Java, desarrollado y adaptado especialmente para el centro **CIDE (Centre Internacional d'Educació)** con motivo de la celebración de la **Fiesta CIDE**.

## 🎮 Sobre el Proyecto

El juego es una aventura clásica de plataformas, con varios mapas, recolección de puntos, enemigos y físicas de gravedad. Todo el desarrollo se apoya en las librerías gráficas estándar de Java (`java.awt`).

### Características Principales:

- Motor de físicas y colisiones personalizado.
- Animación de Sprites y carga de mapas desde archivos de texto.
- Desarrollado usando NetBeans pero compilable desde línea de comandos.

## 🚀 Cómo Jugar

Asegúrate de tener Java instalado en tu sistema. Puedes ejecutar el juego de dos formas:

1. Haciendo **doble clic** sobre el archivo `SuperMiroGame.jar` en la carpeta raíz.
2. Desde la línea de comandos o terminal, ejecutando:
   ```bash
   java -jar SuperMiroGame.jar
   ```

## 🛠️ Estructura del Código

- `src/`: Contiene todo el código fuente del juego.
- `images/`: Recursos visuales, sprites y fondos.
- `maps/`: Archivos `.txt` con el diseño de los niveles (usando caracteres ASCII).
- `SuperMiroGame.jar`: Versión ejecutable lista para jugar.
